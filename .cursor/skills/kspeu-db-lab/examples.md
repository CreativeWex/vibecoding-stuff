# Примеры

## Пример 1 — скилл применяется

**Запрос:** «Выполни лабораторную работу по БД по приложенной методичке, база в dump.sql».

**Вложения:** `postgresql_course.pdf`, `mydb.sql`.

**Ожидаемое поведение:** проверка активации пройдена → `lab_pipeline.py` → агент строит `steps.json` (примеры из PDF переложены на **таблицы только из** `mydb.sql` / `schema.json`, без посторонних `CREATE TABLE` / `lab_*`; в `notes` для многосессионных примеров — **пошаговая** инструкция с **Терминал 1 / Терминал 2** и порядком команд) → `docx_build.py` → пользователю отдаётся `БД_Лабораторная_работа.docx` (без последующего автоформатирования Word).

## Пример 2 — скилл не применяется

**Запрос:** «Как написать JOIN в PostgreSQL?»

**Вложения:** нет или только один файл.

**Ожидаемое поведение:** не открывать KSPEU DB LAB; ответить как в обычном режиме.

## Пример 3 — только SQL, без лабораторной формулировки

**Запрос:** «Разбери этот скрипт» + `.sql` + `.pdf`.

**Ожидаемое поведение:** нет явной просьбы о лабораторной — скилл **не** навязывать; при желании пользователь может уточнить: «сделай лабу по БД».
