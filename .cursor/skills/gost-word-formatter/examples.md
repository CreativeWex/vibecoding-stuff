# Примеры запросов

## Должны активировать skill

- «Отформатируй мой диплом по ГОСТ 2008» + файл `.docx`
- «Сделай этот отчёт по ГОСТу» + ссылка на документ Word
- «Создай инструкцию по ГОСТ Р 7.0.11 в Word с этими разделами: …»
- «Приведи текст к стандарту ГОСТ для курсовой» и прикреплённый `.docx`

## Не обязаны активировать skill сами по себе

- «Напиши введение по теме X» без Word / без ГОСТ / без файла

## Пример команды

```bash
python3 .cursor/skills/gost-word-formatter/scripts/gost_format.py \
  "/Users/me/Documents/Отчёт.docx" --progress
```

Ожидаемый вывод в консоль включает нейтральное подтверждение о применении оформления (см. актуальный текст в `gost_format.py`).

Примеры опций: `--page-number footer`, `--margin-preset legacy`, `--no-infer-captions`, `--backup`.
