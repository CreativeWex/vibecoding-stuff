# ГОСТ Word Formatter (вёрстка под типовую методичку / ГОСТ Р 7.0.11-2008)

## Когда применять

- **Word**, **docx**, **документ Word**, **ГОСТ**, **ГОСТ Р 7.0.11**, запросы **отформатировать / оформить / привести к ГОСТу**, вложения **`.docx` / `.docm`**.

**Не** считать триггером одно «напиши текст» без Word и без оформления.

## Честный scope

**ГОСТ Р 7.0.11-2008** формально про **библиографическое описание**; требования к полям и колонтитулам в отчётах часто берут из **методички вуза**. Скрипт даёт **согласованную вёрстку** с переключаемыми пресетами (см. [reference.md](reference.md)), а не «юридически полный» ГОСТ за один прогон.

## Установка

```bash
python3 -m pip install -r .cursor/skills/gost-word-formatter/scripts/requirements.txt
```

Skill: папка `gost-word-formatter` в `.cursor/skills/` проекта или в `~/.cursor/skills/`.

## Запуск

```bash
python3 .cursor/skills/gost-word-formatter/scripts/gost_format.py "/path/to/file.docx"
```

Результат: **`ГОСТформат<имя>.docx`**. Резервная копия **`*.gost_backup`** только при флаге **`--backup`**.

### Важные флаги

| Флаг | Назначение |
|------|------------|
| **(по умолчанию)** | Подписи к **рисункам** (под рисунком) и **таблицам** (над таблицей), если их не было: краткий текст из контекста, формат `Рисунок N — Название` / `Таблица N — Название`, длинное тире, без точки в конце названия; затем сквозная перенумерация. |
| `--no-infer-captions` | Не добавлять и не генерировать подписи автоматически. |
| `--backup` | Сохранить копию исходного файла с суффиксом `.gost_backup`. |
| `--page-number header` | **По умолчанию** — номер сверху по центру (частые сводки). |
| `--page-number footer` | Номер снизу (старый вариант). |
| `--margin-preset methodology` | **По умолчанию** — поля **25 / 10 / 20 / 20** мм. |
| `--margin-preset legacy` | Поля **30 / 15 / 20 / 20** мм. |
| `--margins-mm L R T B` | Явные поля, мм. |
| `--no-preserve-center` | Не сохранять исходное центрирование титула. |
| `--no-lab-headings` | Отключить эвристику «Цель», «Задание», «Ход работы», … |
| `--toc` / `--no-toc` | Принудительно оглавление / без него (иначе авто при **>3** заголовков по стилю **или** эвристике). |
| `--progress` | Прогресс в stderr на больших файлах. |

После открытия в Word: **обновить поля** (TOC, PAGE).

## Workflow агента

1. **`.doc`** → попросить сохранить как **`.docx`**.  
2. Запустить скрипт с `--progress` на длинных документах; подписи к рисункам/таблицам по умолчанию **генерируются из контекста** (при необходимости отключить: **`--no-infer-captions`**).  
3. Сообщить путь к **`ГОСТформат…docx`**; упомянуть **`--backup`**, только если пользователь просил сохранить копию исходника.  
4. Подтверждение без завышения: например **«Оформление применено (см. консоль и reference.md); сверьте с методичкой кафедры»** — в консоли скрипт печатает нейтральную формулировку.

## Ограничения

- Библиография: правка **текста** записей отдельно.  
- Сложные секции, макросы `.docm`, альбомные страницы — возможна ручная доводка в Word.
