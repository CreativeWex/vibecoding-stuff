# Параметры вёрстки (skill reference)

Сводка того, что делает `scripts/gost_format.py`. **ГОСТ Р 7.0.11-2008** в юридическом смысле про **библиографическое описание**; типовые требования к полям, колонтитулам и шрифтам в вузах часто **сводятся к отдельной методичке** — skill даёт настраиваемый «базовый» профиль и не заменяет её текст.

## Макет страницы

| Параметр | По умолчанию (`--margin-preset methodology`) | Альтернатива (`--margin-preset legacy`) |
|----------|-----------------------------------------------|----------------------------------------|
| Бумага | A4 (210×297 мм) | то же |
| Левое / правое | **25 мм / 10 мм** | 30 мм / 15 мм |
| Верх / низ | 20 мм / 20 мм | то же |

Явный override: `--margins-mm L R T B` (мм), перекрывает preset.

## Нумерация страниц

| Режим | Поведение |
|-------|-----------|
| **`--page-number header`** (по умолчанию) | Номер **в верхнем** колонтитуле, по центру, TNR 12 pt; на **1-й** странице номер не выводится (отдельный первый колонтитул). Текст в верхнем колонтитуле титула **не затирается**. |
| **`--page-number footer`** | Номер **внизу** по центру (прежний вариант skill). |

После сохранения в Word: обновить поля (F9).

## Шрифты и абзацы

| Элемент | Правило |
|---------|---------|
| Основной текст | Times New Roman 14 pt, 1,5, по ширине, красная строка 1,25 см |
| Заголовки 1–3 | Как в стилях `Heading 1–3` / «Заголовок 1–3» |
| Эвристика разделов | Короткие строки вроде «Цель», «Задание», «Ход работы», «Вывод» без стиля Heading оформляются **как заголовок 2** (`--no-lab-headings` отключает) |
| Титул / центр | Абзацы, которые в исходнике были **по центру**, сохраняются **без красной строки** и с центрированием (`--no-preserve-center` отключает) |
| Подпись к рисунку | 12 pt курсив, по центру; шаблоны `Рисунок N`, **`Рис. N`**, `Figure N` |
| Подпись к таблице | 12 pt, по центру; `Таблица N`, **`Табл. N`**, `Table N` |
| Подписи без текста | **По умолчанию:** для абзаца с рисунком — подпись **под** рисунком; для таблицы — **над** таблицей. Название **эвристически** из соседнего абзаца или первой строки таблицы (`Структура таблицы «…»`). Формат: **`Рисунок N — Название`**, **`Таблица N — Название`** (длинное тире U+2014). Нумерация рисунков и таблиц — **отдельные сквозные** счётчики в порядке тела документа. Отключить: `--no-infer-captions`. |

## Оглавление

Вставляется поле TOC, если суммарно **больше 3** заголовков: стили Heading **или** сработавшая эвристика разделов (в основном тексте, не в центрированных блоках). Флаги `--toc` / `--no-toc` переопределяют.

## Библиография

Нормализация **текста** записей (шаблон `Автор. Название // Журнал. — Год. — №. — С. X–Y`) — вручную или через агента; скрипт задаёт в основном **оформление абзаца**.
